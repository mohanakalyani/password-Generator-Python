# Password Generator

A Python application that generates secure random passwords based on user requirements.

## Features
- User chooses the number of letters
- User chooses the number of symbols
- User chooses the number of numbers
- Random password generation
- Password characters are shuffled for better security

## Technologies Used
- Python
- Random Module

## How to Run

1. Download the project
2. Run:

python password_generator.py
